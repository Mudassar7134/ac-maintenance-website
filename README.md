# موقع صيانة المكيفات — الرياض ❄️

موقع تعريفي من صفحة واحدة لشركة متخصصة في **صيانة المكيفات** في مدينة الرياض.
الواجهة بالكامل باللغة **العربية** مع تخطيط **RTL** (من اليمين إلى اليسار).

A premium, single-page Arabic (RTL) website for an air-conditioning maintenance
company in Riyadh, Saudi Arabia.

---

## ⚙️ التقنيات المستخدمة / Tech Stack

- **React 18 + Vite + TypeScript**
- **Tailwind CSS** (with `dir="rtl"` enabled on `<html>`)
- **GSAP + ScrollTrigger** — كشف الأقسام عند التمرير + بارالاكس
- **Framer Motion** — حركات الدخول، الكاروسيل، والتفاعلات الدقيقة
- خطوط عربية: **Cairo** (العناوين) و **Tajawal** (النصوص) من Google Fonts

## 🚀 التشغيل / Getting Started


## 🧩 الأقسام / Sections

1. **شريط تنقل ثابت** — مع خط سفلي متحرك عند المرور وقائمة جوال.
2. **القسم الرئيسي (Hero)** — العنوان، خلفية تبريد متحركة، زرّا «اتصل الآن» و«واتساب».
3. **خدماتنا** — بطاقات صيانة المكيفات (مخفية، مركزية، سبليت، أنظمة متطورة، تنظيف وتعقيم، شحن فريون وفحص دوري).
4. **لماذا نحن** — مميزات + عدّادات إحصائية متحركة.
5. **كيف نعمل** — اتصل بنا ← نحضر للمعاينة ← صيانة احترافية.
6. **آراء العملاء** — كاروسيل بحركة سلسة.
7. **تواصل معنا** — أزرار اتصال وواتساب كبيرة + بيانات التواصل.
8. **التذييل** — روابط وبيانات وحقوق النشر.

## 🖼️ الصور / Images

الصور الحالية **صور حقيقية مجانية** من Pexels و Unsplash (تراخيص تسمح
بالاستخدام التجاري)، وتم اختيارها ومعاينتها يدوياً لتكون متعلقة فعلاً بصيانة
المكيفات والفنيين (تحتاج اتصال إنترنت لعرضها). تتحكم بها كلها من ملف واحد:

```
src/images.ts
```

**لاستبدالها بصورك الخاصة (موصى به لنشاط تجاري حقيقي):**

1. ضع صورك داخل مجلد `public/images/` (مثل `public/images/technician-1.jpg`).
2. في `src/images.ts` بدّل الرابط إلى مسار صورتك، مثلاً:
   ```ts
   heroTechnician: '/images/technician-1.jpg',
   ```

> ملاحظة قانونية: قبل استخدام أي صورة لأغراض تجارية تأكد من ترخيصها.
> أفضل خيار هو استخدام صورك الحقيقية لأعمال الصيانة.

إذا تعذّر تحميل أي صورة (انقطاع الإنترنت مثلاً)، يعرض الموقع تلقائياً بديلاً
أنيقاً بدل ظهور صورة مكسورة.
